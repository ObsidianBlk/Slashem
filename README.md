## Slashem
A top-down endless ghost hunter game made in 3 hours using [Godot 4 - Beta 9](https://godotengine.org/)

### Controls
#### Keyboard
**WASD** - Movement
**SpaceBar** - Attack

#### Gamepad
**Left Thumbstick** - Movement
**(A) Button** - Attack
